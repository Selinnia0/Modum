# Modum

React Native ve Expo kullanılarak geliştirilmiş bir duygu takip mobil uygulamasıdır.

## Kullanılan Teknolojiler

- React Native
- Expo
- TypeScript
- Android Studio Emulator

## Yerelde Çalıştırma

Projeyi klonlayın:

```bash
git clone https://github.com/Selinnia0/Modum.git
cd Modum
npm install
npx expo start
